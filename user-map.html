1. Theme settings

1.1 Logo Setting

Upload logo and favicon, you can customize the logo size for desktop and mobile

1.2 Page Layout Setting

1.2.1 Width Setting

Configure the maximum width of the page, and the safe distance on the left and right sides of the page

1.2.2 Responsive Scaling Setting

Configure the scaling ratio of the page on the tablet and mobile (if you don't accurately configure the mobile size)

1.3 Color Scheme

The theme uses a color scheme system, supports creating multiple color schemes. Each scheme contains the following color definitions:

1.3.1 Basic Color

  • Background Color
  • Title Color
  • Text Color
  • Primary Color
  • Primary Color Hover
  • Border Color
  • Shadow Color

1.3.2 Primary Button and Secondary Button

  • Background Color Text Color
  • Border Color
  • Hover Background Color
  • Hover Text Color
  • Hover Border Color

1.3.3 Input Box

  • Background Color
  • Text Color
  • Border Color
  • Hover Background Color

1.3.4 Product Variant

  • Background Color
  • Text Color
  • Border Color
  • Hover Background Color
  • Hover Text Color
  • Hover Border Color

1.3.5 Selected State

  • Background Color
  • Text Color
  • Border Color
  • Hover Background Color
  • Hover Text Color
  • Hover Border Color

1.4 Font Related

1.4.1 Font Selection

The theme supports 4 font categories:

  • Body Font
  • Subheading Font
  • Heading Font
  • Feature Font

1.4.2 Paragraph Font Setting

  • Font Size
  • Line Height Option
  • Font Weight

1.4.3 Title Font Setting (H1~H6)

  • Font: heading/body/subheading/feature (default: heading)
  • Size: 24-96px (default: 48px)
  • Font Weight: 200-900 (default: 400)
  • Line Height: display-tight/normal/loose (default: tight)
  • Letter Spacing: heading-tight/normal/loose (default: normal)
  • Text Transform: none/uppercase (default: none)

1.5 Button Style

1.5.1 Primary Button (Primary Button) and Secondary Button (Secondary Button)

  • Border Width
  • Border Radius
  • Font Weight
  • Size
  • Font Size

1.6 Shopping Cart Settings

1.6.1 Basic Settings

Shopping Cart Type

  • Page: Full Page
  • Drawer: Drawer (default)

Drawer Color Scheme: Select color scheme (default: scheme-1) - only displayed when Drawer Shopping Cart is selected

Product Title Case

  • Default: Default
  • Uppercase: Uppercase
  • Price Font: body/subheading/heading/feature (default: subheading)
  • Automatically Open Shopping Cart Drawer: Open (default)/Close - automatically open the drawer after adding products
  • Display Shopping Cart Notes: Open (default)/Close
  • Display Discount Code Add: Open (default)/Close
  • Display Installments: Close (default)/Open
  • Display Quick Checkout Button: Open (default)/Close - Apple Pay, Google Pay etc.

1.6.2 Currency Code

Display Currency Code in Shopping Cart Total: Open (default)/Close

1.7 Component Styles

1.7.1 Input Box Style

  • Border Width: 0-4px (default: 2px)
  • Border Radius: 0-32px (default: 12px)

1.7.2 Card Style

Card Border Radius: 0-32px (default: 16px)

1.7.3 Icon Settings

Icon Border Width

  • Thin: Thin
  • Normal: Normal (default)
  • Thick: Thick

1.8 Product Settings

1.8.1 Product Card

  • Quick Add Function: Open (default)/Close - add to cart directly on the product card
  • Mobile Quick Add: Open (default)/Close
  • Display Variant Images: Open (default)/Close - switch product image when selecting variant

1.8.2 Variant Color Card

  • Display Variant Images on Color Card: Open (default)/Close
  • Color Card Width: 20-60px (default: 32px)
  • Color Card Height: 20-60px (default: 32px)
  • Color Card Border Radius: 0-50px (default: 8px)

1.8.3 Price Display

  • Display Currency Code on Product Page: Close (default)/Open
  • Display Currency Code on Product Card: Close (default)/Open
  • Display Currency Code in Shopping Cart: Open (default)/Close

1.9 Badge Settings

  • Display Badge: Open (default)/Close - display "Sold Out", "Promotion" etc. badges
  • Sold Out Badge Color Scheme: scheme-1(default)
  • Promotion Badge Color Scheme: scheme-1(default)
  • Badge Border Radius: 0-50px (default: 12px)

1.10 Search Function

1.10.1 Basic Settings

  • Enable Search: Open (default)/Close
  • Enable Search Modal: Open (default)/Close - modal search interface
  • Search Modal Color Scheme: scheme-1(default)

1.10.2 Layout Settings

  • Desktop Grid Columns: 1-5 columns (default: 3 columns)
  • Mobile Grid Columns: 1-5 columns (default: 2 columns)
  • Search Results: 3-12 (default: 6)

1.10.3 Display Options

  • Display Product Supplier: Close (default)/Open
  • Desktop Media Width: 0-500px (default: 200px)
  • Mobile Media Width: 0-500px (default: 50px)

1.11 Animation Effect

  • Enable Animation: Open (default)/Close - global animation switch
  • Animation Duration
    • Fast: Fast
    • Normal: Normal (default)
    • Slow: Slow

3. Footer Section

Optional Configuration: Color Preset; menu selection; desktop row/column spacing, mobile row/column spacing

3.1.1 Block Settings

Button

Optional Configuration: button text, redirect link, whether to enter a new page

Text

Optional Configuration: text content, using H1 tag (Use with caution! It will convert HTML tags into H1 tags, and using it carelessly will cause SEO issues).

Menu (Footer menu)

Optional Configuration: Menu links, Menu top title, whether to display mobile fold box icon, whether to automatically expand on mobile, title and menu link text font preset

Social Medias

New Social Media block, can configure link, Icon will automatically display according to the type of link.

Column (can configure containers arranged vertically)

Optional Configuration: arrangement direction, container length/height selection, horizontal/vertical alignment, desktop/mobile sub-element spacing.

4. Custom Section

Default Configuration: page width, container height, color preset, background image configuration, desktop/mobile top and bottom spacing.

4.1 Block Settings

Can configure all common blocks in the current theme.

5. Marquee

Default Configuration: page width, container height, color preset, background image configuration, desktop/mobile top and bottom spacing.

5.1 Block Settings

Can configure all common blocks in the current theme.

6. Media Group

Default Configuration: color preset, desktop/mobile top and bottom spacing.

6.1 Block Settings

Row/Column (Same as before)

Media Group

Optional Configuration: color preset; divide the current container into several columns; space between mobile/desktop images; whether to use custom configuration for the height of sub-element images (if yes, the base height of the sub-element container is configurable, if 100px is configured, then the height of the sub-container is the value of Media Column multiplied by 100px; if no, then the base height of the sub-container is the value of the width of each column: approximately 1:1); desktop/mobile top and bottom spacing.

Sub-Block (Media Group Item)

Optional Configuration: mobile/desktop media image and video configuration, and image redirect link; color preset configuration, border radius; desktop/mobile can configure how many rows and columns (the number of columns is calculated according to the number of columns into which the Media Group is divided, and the number of rows is calculated according to the base height multiplied by the preset value).

7. Shop the Look

Default Configuration: color preset, desktop/mobile top and bottom spacing.

7.1 Block Settings

Shop the Look

Optional Configuration: banner image and product carousel layout ratio; space between desktop and mobile two containers; product display configuration, product List and collection two can choose one (list weight is higher than collection; please note: the number of Shop the Look Button needs to be the same as the number of products)

Sub-Block (Shop the Look Points)

Optional Configuration: mobile/desktop image configuration, mobile/desktop border radius configuration, whether to render the height of the image container according to the ratio of the image (if no, the height of the mobile/desktop image container can be customized); desktop/mobile top and bottom spacing.

Sub-Block (Shop the Look Button)

Optional Configuration: relative distance from the top and left of the desktop/mobile button, size of the desktop/mobile button, color related to the button.

Sub-Block (Product Card)

Optional Configuration: color preset, border radius, whether to render the product image according to the original ratio (if no, the height of the desktop/mobile container can be customized); desktop/mobile top and bottom spacing.

Sub-Block (Product Title)

Optional Configuration: associated product (default: the product of the current card), other configuration related to Text.

Sub-Block (Product Price)

Optional Configuration: associated product (default: the product of the current card), font style preset.

Sub-Block (Product Form)

Optional Configuration: associated product (default: the product of the current card).

Sub-Block (Product Quantity、Product Add to Cart、Product Checkout)

Sub-Block (Product Variant)

Optional Configuration: associated product, whether to display the name of the variant button, whether to only display the color variant, variant button style.
